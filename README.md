# ML SEO Optimizer - Versión Monetizada (Premium)

Esta es la versión avanzada de la herramienta de optimización SEO para MercadoLibre, la cual incluye el sistema de gestión de tokens y pagos.

## Características de esta versión:
- **Sistema de Tokens**: Control de saldo y consumo por producto generado.
- **Integración con Pagos**: Recarga de tokens mediante Mercado Pago.
- **Búsqueda con IA (Gemini 3 Flash)**: Utiliza herramientas de búsqueda en tiempo real para investigación de palabras clave.
- **Análisis de Imágenes**: Capacidad de leer imágenes para extraer características sin inventar datos.
- **Modo Administrador**: Acceso ilimitado para la cuenta principal (`pconti10@gmail.com`).
